Moving forward, this is the step to html css.

  1. Create a new html and css file.
  2. Start a html template using shortcut (!)
  3. Link the css file to the html file (in head)
  4. Use Live Server to preview page.
  5. Start building !

What is the differences of ol and ul?

  • Organized list (ol) will have numbered list.
  • Unorganized list (ul) will have bullet list.
